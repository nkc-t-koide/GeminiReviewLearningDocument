AIレビューコメントを確認する

PRのConversationに、github-actions bot からAIレビューコメントが投稿されていれば成功です。Summary、Critical、Major、Minor などの形式でレビュー結果が表示されます。

Workflowの成功を確認する

PR画面のChecksで、AI PR Review が成功していることを確認します。All checks have passed と表示されていれば、Workflowは正常に完了しています。

確認するポイント
  • AI PR Review のチェックが成功している
  • github-actions bot からコメントが投稿されている
  • コメントの中に Summary / Critical / Major / Minor / Good Points / Questions が表示されている
  • beginner / normal / pro でレビューの内容や深さが変わっている

AIレビューコメントが投稿されないとき

Workflowが成功しているのにPRコメントが投稿されない場合は、GitHub Actions の権限設定を確認します。

  1. リポジトリの Settings を開く
  2. Actions → General を開く
  3. Workflow permissions を確認する
  4. 「読み取りおよび書き込み権限」を選択する
  5. Save を押す
補足： この設定が読み取り専用のままだと、GITHUB_TOKEN がPRにコメントを投稿できず、GitHub API status: 403 になることがあります。
